css 문법 | one-way UI 애니메이션 기본 틀

  1. 시작 스타일(전체 틀) -> 최종 스타일(변할 속성만) 각 두가지 class 만들기
  2. JS로 원할 때 시작에서 최종으로 스타일 변하도록 만들기
  3. display: none;(아예 지움) -> 애니메이션에는 visibility: hidden;(모습만 숨김)
  4. display: block; -> visibility: visible;
  5. 스타일 | 시작하는 부분에 transition: all 1s; (1초 동안)
  6. overflow: hidden; (넘치는 컨텐츠 숨김) -> 위아래로 늘어나는 효과 가능
